給建築師、室內裝修業者與代辦窗口使用的 MCP 原型。它把程序分流、送件文件、補正公文、消防與材料文件提示拆成可追溯的工具輸出，讓 AI 先查來源、列缺口、問人工確認，而不是直接下結論。

它想減少的工作量

室內裝修送審最耗時的不是單一法條查詢，而是把程序、文件、圖說、補正公文與專業責任分清楚。這個原型把這些工作拆成可重跑、可稽核的輔助流程。

判斷走哪個程序

依已遮罩文字與檔案 metadata 判斷案件比較接近圖說審核、竣工查驗、變更使用併室內裝修竣工查驗或簡易室內裝修，並附信心分數。

整理送件文件

依程序階段產生新北市室內裝修 packet，列出缺件、照片、材料證明與需要人工確認的項目。

拆補正公文

把遮罩後的補正公文拆成 atomic correction items，輸出回覆草稿、專業確認包與補正摘要。

保留來源與日期

法規 snapshot 與工具輸出保留來源 URL、authority rank、license/update policy、as-of date 與 gate metadata。

資料邊界

可以交給工具的內容

  • 已遮罩的公文文字。
  • 檔名、圖號、頁名、sheet type、文件類型等 metadata。
  • 已去識別化 fixture 或測試案例。
  • 人工確認答案、法規 snapshot、acceptance output。

不要直接貼進 prompt

  • 未遮罩姓名、地址、電話、身分證字號。
  • title block、raw drawing、raw PDF、完整申請文件影像。
  • 客戶授權範圍不明的資料。
  • 可被誤用成法律意見或審查保證的結論。

快速開始

1

Clone repo

git clone https://github.com/trionnemesis/cc-crossbeam-tw.git
cd cc-crossbeam-tw
2

啟動 MCP server

Codex App 設定在 .codex/config.toml；Claude Code 設定在 .mcp.json。

python3 scripts/tw_law_mcp_stdio.py
3

先跑驗收

確認本機工具、fixtures、source adapters、scenario matrix 與 two-stage flow 都能重跑；aggregate trust gate 會保留 G2 fail-closed 狀態。

python3 scripts/run_phase_acceptance.py

目前原型狀態

版本 0.4.0 prototype
首發地區 新北市
案型 室內裝修
完成階段 1~6 組 + Phase 2.1-2.6 / Step 6

1. Source policy

P0 法規來源、authority rank、license/update policy 與 fallback comparison。

2. Procedure + HITL

程序階段信心分數、低信心分流與人工確認 loop。

3. G2 fixture contract

12 份 synthetic de-identified cases、84 個 atomic correction items；unsupported claims fail closed。

4. Metadata extraction

metadata-only drawing/document policy、sheet manifest 與 raw content boundary。

5. Jurisdiction registry

新北市 enabled；台北、桃園 stubs fail closed，不主動作答。

6. MCP packaging

Standalone MCP server first，Codex 與 Claude Code 維持 thin wrappers。

主線已完成 1~6 組開發，並完成 Phase 2.1-2.6 / Step 6 的 data layout、source adapters、scenario tools、hardened scenario matrix 與 two-stage contractor flow skeleton。Production 導入前仍需 approved real de-identified cases 與 live official-source ingestion/refresh workflow；G2 synthetic baseline 只驗證 schema、gate 與 HITL contract，不支撐真實案件 claim。

cc-crossbeam-tw 只提供來源可追溯的文件輔助，不提供法律意見、合規保證、違章認定或審查必過承諾。 實務場景主索引見 tw-scenario-feature-matrix.md。
